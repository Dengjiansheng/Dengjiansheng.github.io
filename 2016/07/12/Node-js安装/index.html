Windows懒人安装

  • 安装Windows-nvm，安装前确保电脑里面没有安装其他Node版本，不然有可能会有一些奇怪的问题
  • 直接通过nvm install 4.x.x来安装相应的版本
  • 其他用法与ubuntu上的nvm用法差不多，无非就是nvm ls/nvm use 4.x.x这一类的
